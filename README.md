# Decentralized Online Auction DApp

A highly robust and aesthetic blockchain-based auction system. It allows users to create auctions, seamlessly place bids, and reliably determine winners using Ethereum smart contracts, with zero centralized authorities.

## 🌟 Key Features
- **Decentralized and Trustless**: No middleman required to facilitate auctions.
- **Transparent Bidding**: All bids are recorded permanently and viewable on-chain.
- **Secure Fund Handling**: Smart contracts handle money programmatically. Uses a highly secure "Pull over Push" withdrawal pattern to mitigate smart contract re-entrancy vulnerabilities.
- **Automated Settlement**: When the time expires, anyone can settle the auction to automatically transfer the highest bid to the creator.
- **Premium User Interface**: Built with an elegant glassmorphism aesthetic, responsive design, and smooth animations using TailwindCSS.
- **Live Updates**: Real-time countdowns and on-chain bid evaluations.

## 🛠️ Technology Stack
- **Web3 / Smart Contracts**: Solidity, Hardhat, Ethers.js v6
- **Frontend**: React.js, Vite
- **Styling**: TailwindCSS v4 with custom glassmorphism utilities
- **Wallet Integration**: MetaMask

## 🏗️ Project Architecture
The project is split into two primary directories:

### 1. Backend (`/backend`)
Contains the Solidity smart contracts and the Hardhat configuration.
- `contracts/AuctionManager.sol`: The core logic contract handling multiple concurrent auctions.
- `test/Comprehensive.test.js`: A rigorous test suite guaranteeing the safety of the dApps logic across 18 unique public use scenarios.
- `scripts/deploy.js`: Automates deployment and synchronization of the Contract block address and ABIs straight to the frontend.

### 2. Frontend (`/frontend`)
The user interface built on React & Vite.
- Connects directly to the user's MetaMask wallet.
- Reads global state via a centralized `Web3Context`.
- Fully responsive Dashboard, Live Auction Grid, and Minting workflows.

## 🌍 Deploying to Production (Render.com)

This project is **100% deployment-ready** and includes a `render.yaml` Blueprint file explicitly configured to automate the deployment process.

1. First, deploy your smart contract to a live blockchain (like the **Sepolia Testnet**) and commit the updated `frontend/src/contracts` folder to your code.
2. Push your entire project folder to a new **GitHub repository**.
3. Log in to your [Render Dashboard](https://dashboard.render.com).
4. Click the **"New +"** button in the top right and select **Blueprint**.
5. Connect the GitHub repository you just created.

Render will instantly detect the `render.yaml` file, build the Vite app, configure the routing rules, and deploy your frontend globally on a CDN!

## 🚀 Getting Started (Run Locally)

### Prerequisites
Make sure you have installed:
- [Node.js](https://nodejs.org/en/) (v18+)
- [MetaMask Browser Extension](https://metamask.io/)

### 1. Setup the Smart Contract (Backend)
Open a terminal and navigate to the backend directory:
```bash
cd backend
```

Install Hardhat and dependencies:
```bash
npm install
```

Start the local Hardhat blockchain node. **Keep this terminal window running**:
```bash
npx hardhat node
```
*(Hardhat will generate 20 test accounts loaded with fake 10,000 ETH. Import one of these private keys into your MetaMask wallet to interact with the dApp).*

Open a **new terminal tab** (still in the `backend` directory) and deploy the smart contract to your local node:
```bash
npx hardhat run scripts/deploy.js --network localhost
```
*(This script will also automatically copy the required `AuctionManager.json` ABI and contract address to the frontend folder).*

### 2. Setup the React App (Frontend)
Navigate to the frontend directory:
```bash
cd ../frontend
```

Install frontend dependencies:
```bash
npm install --legacy-peer-deps
```

Start the Vite development server:
```bash
npm run dev
```

Visit the application at: **http://localhost:5173**

### 3. Connect to MetaMask
1. Open your MetaMask browser extension.
2. Select **Add Network** -> **Add a network manually**.
3. Name: `Localhost 8545`
4. RPC URL: `http://127.0.0.1:8545`
5. Chain ID: `31337`
6. Currency Symbol: `ETH`
7. Save and switch to this network.
8. Import one of the private keys generated by `npx hardhat node` into MetaMask to have test funds!

## ✅ Running the Tests
We wrote an exhaustive 18-case test suite covering edge cases, bidding constraints, accumulation of returns, and security patterns.

To run the smart contract tests, navigate to the `backend` directory and run:
```bash
npx hardhat test test/Comprehensive.test.js
```
